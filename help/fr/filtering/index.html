    Filtrage, tri et recherche

    Vous pouvez filtrer les processus et les applications qui ne vous intéressent pas:

    Il y a une longue liste de mots-clés que vous pouvez utiliser pour rechercher dans la liste de processus qui vous préoccupe :

    Et une liste encore plus longue de paramètres que vous pouvez appliquer pour trier la liste des processus :

    • Type : Types définis dans les filtres du processus. Veuillez consulter la capture d'écran ci-dessus.
    • Mémoire : Consommation de RAM
    • Utilisation du processeur / Impact du processeur: Veuillez voir Quelle est la différence entre "Utilisation du processeur" et "Impact du processeur"?
    • I/O: Données lues depuis le stockage et écrites sur le stockage. Voir proc(5).
    • PPID / PID / UID / Âge : ID de processus parent, id de processus, identifiant utilisateur UNIX et âge du processus natif Linux. Voir proc(5).
    • Paquets / Activités / Services : Nombre d'applications, leurs tâches visibles et les services exécutés dans le processus natif.
